============
Usage:
============

1. On terminal run the following commands:
	a. python run.py		//executes the program
	b. python clean.py		//cleans the directory

================
Additional Info:
================

1. Image2PX.jar : Jar file that converts image to pixel values
2. PX2image.jar : Jar file that converts pixel values to image
3. run.py	: script that executes following commands
		: 	a. java -jar Image2PX.jar original_image.jpg original_image.px					//Converting image to input pixel value file 
			b. nvcc parallel_non_optimized.cu -o parallel_non_optimized				 	//Generating cuda executable
			c. ./parallel_non_optimized original_image.px output_image.px decompress_output_image.px	//./<cuda executable> <input pixel file> <output pixel file>
 			d. java -jar PX2Image.jar output_image.px jpg output_image.jpg					//Converting output pixel value file to image
			e. java -jar PX2Image.jar decompress_output_image.px jpg decompress_output_image.jpg		//Converting output pixel value file to image
4. clean.py	: script that deletes all files generated by running run.py
5. original_image.jpg :Image used for input to cuda executable
